# Portfolio - Alejo Carmona

Portfolio personal desarrollado para mostrar mis proyectos, habilidades y experiencia como Frontend Developer.

## Tecnologías utilizadas

- React
- TypeScript
- Vite
- Tailwind CSS
- Framer Motion
- React Icons

## Características

- Diseño moderno y responsive
- Animaciones fluidas con Framer Motion
- Sección de proyectos
- Información de contacto
- Descarga de CV
- Optimización para rendimiento y experiencia de usuario

## 📂 Instalación

```bash
npm install
```

## ▶️ Ejecutar en desarrollo

```bash
npm run dev
```

## 📦 Build de producción

```bash
npm run build
```
